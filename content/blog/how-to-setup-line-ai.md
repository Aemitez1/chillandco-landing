## ก่อนเริ่ม — สิ่งที่ต้องมี

- LINE Official Account (LINE OA) ที่ active อยู่แล้ว
- สมาร์ทโฟนหรือคอมพิวเตอร์
- ข้อมูลร้านพื้นฐาน (รายการสินค้า, ราคา, ช่องทางชำระเงิน)

ไม่ต้องรู้ programming ไม่ต้องเช่า server เองครับ

## ขั้นตอนที่ 1: เพิ่มเพื่อน Chill&Co Works

เริ่มจากการ[เพิ่มเพื่อน LINE OA ของ Chill&Co Works](https://line.me/ti/p/@chillandco) แล้วทีมเราจะส่ง link Dashboard ให้ภายใน 24 ชม.

## ขั้นตอนที่ 2: เชื่อมต่อ LINE OA ของคุณ

ใน Dashboard ไปที่ **Settings → LINE Integration** แล้วกรอก:
- **Channel ID** และ **Channel Secret** จาก LINE Developers Console
- **Channel Access Token** (Long-lived token)

> วิธีหา Token: เข้า [developers.line.biz](https://developers.line.biz) → เลือก Messaging API → Scroll ลงมาหา Channel Access Token

## ขั้นตอนที่ 3: กรอก Knowledge Base

นี่คือส่วนสำคัญที่สุด — ยิ่งกรอกข้อมูลละเอียด AI ยิ่งตอบได้แม่นยำ

**PERSONA (บุคลิก AI)**
```
ชื่อร้าน: [ชื่อร้านคุณ]
ประเภทธุรกิจ: [เช่น ร้านเสื้อผ้า]
สไตล์การตอบ: สุภาพ เป็นกันเอง ลงท้ายด้วย "ค่ะ"
```

**REPO (ข้อมูลสินค้า)**
```
สินค้า A — ราคา 299 บาท — มีสีดำ ขาว น้ำเงิน
สินค้า B — ราคา 499 บาท — Size S M L XL
```

**PROTOCOL (การสั่งซื้อ)**
```
ช่องทางชำระ: โอนบัญชี SCB 123-456-789
ค่าส่ง: 50 บาท ส่ง Kerry ทุกวัน จันทร์-ศุกร์
```

**LOGIC (FAQ)**
```
Q: สั่งแล้วได้รับเมื่อไหร่?
A: สั่งก่อน 12.00 น. ส่งวันเดียวกัน ได้รับใน 1-2 วัน

Q: เปลี่ยน/คืนสินค้าได้ไหม?
A: เปลี่ยนได้ภายใน 7 วัน สินค้าต้องไม่ผ่านการใช้งาน
```

## ขั้นตอนที่ 4: ตั้งค่า Admin Group

เพิ่ม LINE Bot เข้ากลุ่มแอดมินของคุณ เพื่อรับแจ้งเตือนเมื่อ:
- AI ส่งต่อลูกค้าให้แอดมิน (Handover)
- มีออเดอร์ใหม่เข้ามา

## ขั้นตอนที่ 5: เปิดใช้งาน

ไปที่ **Dashboard → AI Core → เปิดใช้งาน** — เท่านั้นเลยครับ

AI จะเริ่มตอบลูกค้าทันที คุณสามารถดู log การสนทนาแบบ Real-time ใน Dashboard ได้

## ตรวจสอบว่าทำงานถูกต้อง

ทดสอบโดยการส่งข้อความทดสอบเข้า LINE OA ของคุณจากโทรศัพท์อีกเครื่อง หรือขอให้เพื่อนลองแชท ควรได้รับการตอบภายใน 4-5 วินาที
